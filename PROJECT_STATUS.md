## 1. 项目概述

### 1.1 项目名称
AI World - AI公司模拟器

### 1.2 项目类型
2D俯视视角剧情类游戏 + AI工作流模拟器

### 1.3 核心玩法
玩家扮演公司老板，通过AI规划师分配任务给员工，员工使用AI API并行完成任务，最终自动合并成果并格式化为Markdown文档。

### 1.4 游戏场景
- **工作区**: 6个工位，员工在此执行任务
- **休息区**: 沙发、绿植，员工休息和等待区域
- **走廊**: 连接工作区和休息区的通道
- **老板**: 玩家控制的角色，可移动并与员工交互

## 2. 游戏内详细内容

### 2.1 员工系统（6个员工）

| ID | 姓名 | 默认职位 | AI模型 | API平台 |
|----|------|---------|--------|---------|
| 0 | 小明 | 技术专员 | qwen-max | 阿里云百炼 |
| 1 | 小红 | 人事专员 | qwen3.6-plus | 阿里云百炼 |
| 2 | 阿强 | 市场专员 | qwen3-vl-235b-thinking | 阿里云百炼 |
| 3 | 小丽 | 财务专员 | qwen-coder-turbo-0919 | 阿里云百炼 |
| 4 | 大伟 | 运营专员 | qvq-max-2025-03-25 | 阿里云百炼 |
| 5 | 晓晓 | 行政专员 | qwen-plus | 阿里云百炼 |

**员工状态**:
- 工作中（在工位上）
- 休息中（在休息区）
- 行走中（移动中）
- 坐着（休息区坐着）

### 2.2 工作模式

**仅支持并行模式**:
- 所有选中的员工同时开始工作
- 各自独立完成任务
- 无任务依赖关系
- 各自完成后立即返回休息区

### 2.3 AI规划流程

```
用户输入需求 + 上传文件/图片
    ↓
AI规划师分析需求（使用最强模型）
    ↓
为每个员工生成角色分工
    ↓
生成技能文件(skill/skillX.md)
    ↓
用户按空格发布号令
    ↓
员工前往工位
    ↓
并行执行AI任务（各自使用不同模型）
    ↓
实时显示进度
    ↓
任务完成，保存成果
    ↓
AI输出处理器严格处理每个输出
    ↓
缝合所有员工结果
    ↓
规划师整合并格式化为Markdown
    ↓
生成MERGED_RESULT.md
    ↓
员工返回休息区
```

### 2.4 技能文件格式

每个员工在任务开始前会生成技能文件 `skill/skillX.md`:

```markdown
---
name: 员工姓名
role: 角色名称
employee_id: 0
mode: 并行
generated_at: 2026-04-10 10:30:00
generated_by: AI World Agent Planner V2
---

# 员工姓名 - 角色名称

## 身份定位
[AI规划师生成的角色描述]

## 工作内容
1. [具体任务1]
2. [具体任务2]

## 输入要求
- 前置信息：[需要什么输入]

## 输出标准
- 产出物：[输出什么内容]
- 格式规范：[输出格式]

## 工作方法
- 分析思路：[如何分析]
- 执行步骤：[具体步骤]

---
*This skill file was automatically generated by AI World Agent Planner V2*
```

## 3. 完整操作流程

### 3.1 首次运行

1. **启动游戏**
   ```bash
   cd C:\Users\admin\Desktop\AI-World
   py main.py
   ```

2. **员工API已自动配置**
   - 游戏启动时自动为6名员工分配最优模型
   - 阿里云百炼API已预配置
   - 智谱GLM作为备用

### 3.2 创建项目

1. **打开流程编辑器**
   - 按 `P` 键
   - 或走到员工附近交互

2. **选择员工**
   - 点击员工按钮（显示绿色边框为选中）

3. **输入需求**
   - 在输入框中输入提示词
   - 支持中文输入、复制粘贴
   - 可上传文件/图片作为参考

4. **AI规划**
   - 点击"开始规划"
   - AI规划师分析需求
   - 生成分工方案和技能文件
   - 显示每个员工的角色

5. **发布号令**
   - 按 `空格` 键
   - 员工自动前往工位
   - AI工作器开始并行执行

### 3.3 监控进度

1. **查看工作流面板**
   - 按 `T` 键切换显示
   - 拖动标题栏移动面板
   - 滚轮滚动查看所有任务

2. **面板内容**
   - 整体进度条（顶部）
   - 每个任务的实时进度
   - 进行中任务显示迷你进度条
   - 已完成任务显示耗时

### 3.4 查看成果

1. **任务完成**
   - 员工自动返回休息区
   - AI输出处理器自动清理格式
   - 规划师整合所有结果为Markdown

2. **成果文件位置**
   ```
   results/
   └── [公司名称]_[项目名]_[时间戳]/
       ├── README.md              # 项目报告
       ├── MERGED_RESULT.md       # 整合成果（格式化Markdown）
       ├── project_meta.json      # 项目元数据
       ├── 001_[角色1].md         # 员工1成果
       ├── 002_[角色2].md         # 员工2成果
       └── ...
   ```
