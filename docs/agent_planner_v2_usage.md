# Agent Planner V2 使用指南

## 概述

Agent Planner V2 是增强版AI规划师，支持：
- 📎 **文件/图片上传** - 支持图片和文档作为输入
- 📝 **结果整合** - 自动整合所有员工结果为统一Markdown文档
- 🎯 **自动创建技能文件** - 为每个员工生成标准化的skill.md

## 使用方法

### 1. 基本使用

```python
from ai_systems.agent_planner_v2 import agent_planner_v2

# 设置API配置（使用阿里云百炼）
agent_planner_v2.set_config(
    api_key="sk-7c497750be384766a027179211f1c010",
    api_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
    model="qwen-max"
)

# 生成规划
agent_planner_v2.generate_plan(
    selected_employees=employees,
    user_requirement="创建一个电商网站",
    on_token=lambda token: print(token, end=""),
    on_complete=lambda plan: print("规划完成!")

```

### 2. 上传文件/图片

```python
# 上传图片
agent_planner_v2.upload_file("path/to/design.png")
agent_planner_v2.upload_file("path/to/requirements.jpg")

# 上传文档
agent_planner_v2.upload_file("path/to/specs.txt")
agent_planner_v2.upload_file("path/to/data.json")

# 查看上传摘要
print(agent_planner_v2.get_upload_summary())
# 输出: 已上传 4 个文件:
#   - design.png (image, 245.3KB)
#   - requirements.jpg (image, 189.2KB)
#   - specs.txt (document, 12.5KB)
#   - data.json (document, 5.1KB)

# 清空上传
agent_planner_v2.clear_uploads()
```

### 3. 完整流程示例

```python
from ai_systems.agent_planner_v2 import agent_planner_v2

# 配置API
agent_planner_v2.set_config(
    api_key="sk-7c497750be384766a027179211f1c010",
    model="qwen-max"
)

# 上传参考文件
agent_planner_v2.upload_file("docs/design_mockup.png")
agent_planner_v2.upload_file("docs/requirements.md")

# 步骤1: 生成规划
def on_planning_token(token):
    print(token, end="", flush=True)

def on_planning_complete(plan):
    if plan:
        print(f"\n规划完成! 项目目标: {plan['project_goal']}")
        print(f"已为 {len(plan['employees'])} 名员工生成技能文件")
        
        # 步骤2: 员工开始工作（模拟）
        employee_results = []
        for emp_plan in plan['employees']:
            # 这里应该是实际的工作流程
            result = {
                'employee_id': emp_plan['employee_id'],
                'employee_name': emp_plan['employee_name'],
                'content': f"{emp_plan['employee_name']} 的工作成果..."
            }
            employee_results.append(result)
        
        # 步骤3: 整合结果
        agent_planner_v2.consolidate_results(
            plan=plan,
            employee_results=employee_results,
            output_dir="results/project_001",
            on_token=lambda t: print(t, end=""),
            on_complete=lambda result: print("\n项目完成!")
        )

# 开始规划

```

## 文件上传支持

### 支持的图片格式
- `.png` - PNG图片
- `.jpg`, `.jpeg` - JPEG图片
- `.gif` - GIF动图
- `.bmp` - BMP图片
- `.webp` - WebP图片

### 支持的文档格式
- `.txt` - 纯文本
- `.md` - Markdown文档
- `.json` - JSON数据
- `.csv` - CSV表格
- `.pdf` - PDF文件（会提示描述内容）
- `.doc`, `.docx` - Word文档（会提示描述内容）

## 与旧版本对比

| 功能 | AgentPlanner (旧) | AgentPlannerV2 (新) |
|------|-------------------|---------------------|
| 文件上传 | ❌ 不支持 | ✅ 支持图片和文档 |
| 结果整合 | ⚠️ 简单合并 | ✅ 格式化为标准Markdown |
| 技能文件 | ⚠️ 基础格式 | ✅ YAML frontmatter + 标准格式 |
| 流式输出 | ✅ 支持 | ✅ 支持 |
| 多模态 | ❌ 不支持 | ✅ 支持图片理解 |

## 注意事项

1. **图片大小** - 建议上传的图片不超过2MB，以提高处理速度
2. **文档长度** - 长文档会自动截断到10000字符
3. **API支持** - 文件上传需要模型支持多模态（如qwen-vl系列）
4. **格式标准** - 所有输出都经过Markdown标准化处理
